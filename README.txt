I denne github repository, findes Python scriptet til løsningen af Opgave 2 i den Obligatoriske opgave for kurset i Lineær Optimering.
Python koden er inspireret af Gurobis løsning på problemet, som findes på dette link:
https://www.gurobi.com/resources/food-manufacture-i/
Koden er blevet ændret navnmæssigt og bearbejdet for at passe til min besvarelse.

I denne github repository findes også input samt output fra min Jupyter Notebook. Denne findes i .html form samt .tex form.
For at genskabe resultater, er det nødvendigt at indsætte koden i Jupyter Notebook.
Skrevet af Jacob Qazizada.
